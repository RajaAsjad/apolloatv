# Apollo ATVs — Premium Powersports Distributor Website

Modern, performance-optimized website for [Apollo ATVs](https://www.apolloatvs.com) — wholesale powersports distributor in the USA.

## Features

- **Dealer-focused conversion** — Become a Dealer CTAs throughout, dealer application form
- **Product catalog** — Category & engine-size filtering, detailed specs, color selection
- **Premium UX** — Dark theme, responsive design, trust badges, featured products
- **Performance** — Next.js 16, static generation, optimized assets
- **SEO foundation** — Schema markup, proper headings, metadata, breadcrumbs

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

```bash
npx vercel --prod
```

Or connect the GitHub repository to Vercel for automatic deployments.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage with hero, featured products, categories |
| `/products` | Full catalog with filters |
| `/products/[slug]` | Product detail with specs & gallery |
| `/become-a-dealer` | Dealer benefits & application form |
| `/about` | Company story & values |
| `/contact` | Contact form & info |
